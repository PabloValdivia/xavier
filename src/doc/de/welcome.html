Willkommen bei Xavier!

Ihre erste Analyse?

So einfach geht es:

  1. Links sehen Sie eine Liste lesbare Quadern. Klicken Sie auf der Name einer davon. Dadurch werden Werte und Dimensionen der gewählten Quader gezeigt und die Menuleiste wird geändert.
  2. Der nächste Schritte erstellt und wechselt zu einem neuen Reiter - der 'Willkommen'-Reiter (mit diesem Text) verbleibt aber.
  3. In der Menuleiste klicken Sie auf Kreuztabelle. Die Ansicht wechselt in einen neu erstellten Reiter 'Kreuztabelle'.
  4. Ziehen Sie einen der angezeigten Werte auf dem Wort 'Spalten'. Die Summe dieser Wert über den ganzen Quader wird berechnet und in einer Tabelle angezeigt.
  5. Expandieren Sie die erste Dimension (Klicken Sie auf dem '+'-Zeichen). Sie sehen die Ebenen der Dimension.
  6. Ziehen Sie die erste Ebene auf dem Wort 'Zeilen'. Die Tabelle wird mit einer Zeile erweitert. Klicken Sie auf das Plus-Zeichen, um mehr Details zu sehen.

Weitere Hinweise:

Bitte beachten Sie die kleinen Pfeile links. Damit können Sie u.a. die Liste der Datenquader neu aufrufen.

Wenn Sie ein zu ziehendes Element ausgewählt haben, leuchten mögliche Ziele grün auf. Muss-Elemente sind Fett geschrieben. Muss-Ziele sind mit einem gelben Dreieck markiert. Bei einem Doppelclick auf einem Element zieht Xavier das Element in ein plausibles Ziel.

Sie können mehrere Reiter für einen Datenquader erstellen und bearbeiten.

Sie können die Abfrage eines Reiters speichern oder drucken.

Sie können auch Ihre Ergebnisse nach Microsoft Excel exportieren.

Hilfe Direkt

Für viele Elemente und Symbole ist eine Erklärung verfügbar. Die Erklärung wird angezeigt, wenn der Cursor eine kurze Weile über dem Symbol ruht.

Wenn Sie ein Informations-Icon sehen, sind Informationen verfügbar. Klicken Sie auf das Icon, um die Informationen in einem neuen Reiter zu sehen.

Über Xavier:

Xavier ist ein Werkzeug für Analyse und Visualisierung (OLAP). Xavier arbeitet auf der Basis von XML/A (XML for Analysis). Xavier kann mit allen gängigen XML/A-Servern arbeiten, wie Mondrian, Pentaho, icCube, Microsoft Analysis Services und noch anderen. Die Nutzung von Begriffe und Symbole wird hier erklärt.
